# Descrição

Este projeto foi desenvolvido para a disciplina de Projeto Integrador com o objetivo de criar um sistema simples para o gerenciamento de pacotes turísticos da Agência Destino Certo.

O sistema permite cadastrar, consultar, alterar e excluir pacotes turísticos, facilitando a organização das informações da empresa.

## Funcionalidades

- Cadastro de pacotes turísticos;
- Consulta de pacotes cadastrados;
- Alteração de informações dos pacotes;
- Exclusão de pacotes;
- Menu principal para navegação.

## Tecnologias Utilizadas

- Linguagem C (C99)
- Draw.io (diagramas)
- PDF (documentação)

## Como executar

1. Compile o arquivo `codigo.c`.
2. Execute o programa.
3. Utilize o menu para acessar as funcionalidades do sistema.

## Objetivo

Centralizar as informações da agência, facilitar o atendimento aos clientes e tornar o gerenciamento dos pacotes turísticos mais rápido e organizado.
